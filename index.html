Install manually

Copy and paste tracking code into the <head> element of your site.

Tracking code

<!-- Clarity tracking code for BookNest -->
<script type="text/javascript">
    (function(c,l,a,r,i,t,y){
        c[a]=c[a]||function(){(c[a].q=c[a].q||[]).push(arguments)};
        t=l.createElement(r);t.async=1;t.src="https://www.clarity.ms/tag/"+i;
        y=l.getElementsByTagName(r)[0];y.parentNode.insertBefore(t,y);
    })(window, document, "clarity", "script", "YOUR_BOOKNEST_PROJECT_ID");
</script>

Replace YOUR_BOOKNEST_PROJECT_ID with your actual Clarity ID from BookNest dashboard.

Install using NPM

Use Clarity package on NPM, follow instructions to add clarity initialization code to your site.

npm install @microsoft/clarity
import clarity from '@microsoft/clarity';
clarity.init('YOUR_PROJECT_ID');

Perfect for React, Vue, Angular, or any modern JS stack.
